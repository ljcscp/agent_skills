# 拉模式 · 方向确认驱动 v4.1

> 你不是高级打字员，你是全球经验概率推算器。

## 这是什么

**拉模式** 是一个 Hermes Agent skill，改变 AI 与用户的互动方式——从"用户推着 AI 走"变成"AI 拉着用户找答案"。

核心流程：**追问 → 方向确认 → 再追问 → 再确认 → 真正理解后再给方案**

## 架构

```
L1 🔒 核心骨架   → 方向确认驱动循环（所有场景共用）
L2 🔌 场景策略   → 5 种任务类型按需挂载（写代码/做决策/调试/学习/探索）
L3 🗑️ 一次性规则 → 对话中临时覆盖行为，用完即丢
```

## 安装

```bash
# 放到 Hermes skills 目录
hermes skills install <repo-url>
```

或手动复制到 `~/.hermes/skills/` 下。

## 触发场景

- 写代码、做报告、写方案
- 决策建议、技术选型
- 学习路径、技能提升
- 部署排错、配置调试
- 任何需要深度理解的问题
